## 🚀 Featured Project — LogoCraft

> **AI-Powered Logo Generation SaaS** · [Live →](https://logocraft.nitindogra.space) · [GitHub →](https://github.com/nitindogra7)

| Metric | Result |
|:---|:---|
| 🔴 API Response (Before) | 420ms |
| 🟢 API Response (After Redis) | **140ms** |
| ⚡ Latency Reduction | **62.7% faster · 2.5× speed** |

**What's under the hood:**

```
🔐 Auth      →  Stateless JWT + rotating refresh tokens (HTTP-only cookies)
                Expiry-aware middleware + secure session invalidation on logout

⚡ Cache     →  Redis on dashboard APIs → 420ms dropped to 140ms

💳 Payments  →  Razorpay: server-side order creation + HMAC signature verification
                No client trust. Wallet credited only after validated transaction.

🎨 AI        →  Google Imagen + Cloudinary async pipeline
                Generate → Upload → Secure URL delivery → Persistent history

🏗️ Arch      →  MVC: routing / controller / model / service layers
                CORS allowlisting + .env secrets + environment-specific cookie policies
```

**Stack:** `Node.js` `Express.js` `MongoDB` `Redis` `React` `JWT` `Razorpay` `Cloudinary` `Vercel` `Render`
