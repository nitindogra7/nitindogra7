### 🔩 Architecture Breakdown

```
🔐  AUTH SYSTEM
    ├── Stateless JWT (short-lived access tokens)
    ├── Rotating refresh tokens via HTTP-only cookies
    ├── Expiry-aware middleware
    └── Secure session invalidation on logout

⚡  PERFORMANCE
    ├── Redis caching on all dashboard APIs
    └── 420ms → 140ms  (62.7% latency reduction)

💳  PAYMENT INFRASTRUCTURE
    ├── Razorpay server-side order creation
    ├── HMAC-SHA256 signature verification
    └── Wallet credited ONLY after trusted validation

🎨  AI PIPELINE
    ├── Google Imagen — logo generation
    ├── Cloudinary — async upload + CDN delivery
    └── Persistent user history tracking

🏗️  BACKEND ARCHITECTURE
    ├── MVC: routing → controller → service → model
    ├── Modular middleware + protected routes
    ├── .env secrets + CORS allowlisting
    └── Environment-specific cookie policies
```

**Tech:** `Node.js` `Express.js` `MongoDB` `Mongoose` `Redis` `React` `Tailwind CSS` `JWT` `Razorpay` `Cloudinary` `Google Imagen` `Render` `Vercel`
